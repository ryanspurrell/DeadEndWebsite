Dead End Website

To run this website locally:

1. Download the github repository:

    Without terminal:
        Go to the Code section and click "Download as zip"
        Extract the zipped files, where you'll have a directory called "DeadEnd"

    With terminal:
        git clone https://github.com/ryanspurrell/DeadEndWebsite.git

2. Install Node.js and npm:

    Without terminal:
        Go to nodejs.org/en/download/ to download and install

    With terminal:
    In arch:
        sudo pacman -S nodejs
    In ubuntu:
        sudo apt install nodejs

Everything from this point has to be done in your computer's terminal from the DeadEnd directory

4. Install dependencies:
    npm install

5. Host server locally:
    npm run dev
    The terminal will output a local ip address, either click on it to open the website in a browser or copy paste into browser

6. It should now be running!
